# PROPOSED UPDATES TO NTCIP 8001 {.annex}

1. Should there be observing members for the JC? How do these differ from observing members for WGs? Maybe we should have memberships as follows: voting, non-voting, observational and liaison. In this case, observational would not be given tasks while voting and non-voting members would. Non-voting members would be those that attend regularly but do not have voting privileges.
2. Procedures for reaffirming and rescinding documents should be added in. Note that amended and revised documents have been dropped.

Re-affirmed

NTCIP Standards expire after a period of five years unless they are revised or re-affirmed. In order to be re-affirmed, the exact same text of the standard (without any changes) must be re-balloted by the SDOs and approved. Thus, re-affirmed standards have demonstrated a great deal of stability.

Rescinded

If an NTCIP standard is no longer needed, it can be rescinded, either by vote or by allowing the five-year cycle to lapse. A status of rescinded indicates that there is no longer any version of this standard that is recommended for use.
